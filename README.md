Stiftung Warentest veröffentlicht regelmäßig Untersuchungen zu Nahrungsergänzungsmitteln, Blutdruckmitteln und Herz-Kreislauf-Produkten – Cardiotensive selbst wurde aber noch nicht im Detail geprüft.
